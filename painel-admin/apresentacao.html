Bem-vindo ao Painel Administrativo

Este painel permite gerenciar todas as informações do sistema. Utilize o menu superior para navegar entre as seções. Veja abaixo o que cada opção representa:

📂 Categorias

Cadastre as categorias disponíveis no evento, como “Infantil”, “Adulto”, “Melhor Fantasia”, entre outras. Os participantes serão vinculados a essas categorias.

🧍 Participantes

Cadastre os participantes que irão concorrer em uma ou mais categorias. Cada participante pode ser avaliado pelos jurados com base nos critérios definidos.

🧑‍⚖️ Jurados

Gerencie os jurados responsáveis por atribuir notas aos participantes. Cada jurado tem acesso controlado e só pode votar uma vez por categoria.

✅ Tipos de Avaliação

Configure os critérios de avaliação, como “Beleza”, “Postura”, “Desempenho”, entre outros. Os jurados utilizarão esses critérios para pontuar os participantes.

🏆 Relatório de Ganhadores

Visualize e exporte os resultados finais, com base nas notas atribuídas pelos jurados. Aqui são listados os vencedores de cada categoria.
